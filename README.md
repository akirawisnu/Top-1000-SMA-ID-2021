# Top-1000-SMA-ID-2021
Welcome! This is the simple Python scraper to gather data for Top 1000 SMA/K based on TPS UTBK Score in 2021 based on LTMPT (Lembaga Tes Masuk Perguruan Tinggi) in Indonesia.
